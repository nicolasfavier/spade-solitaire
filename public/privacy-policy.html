Politique de confidentialité - Spider Balala

Dernière mise à jour : 23 janvier 2026

Introduction

Cette politique de confidentialité décrit comment Spider Balala collecte, utilise et protège vos informations personnelles lorsque vous utilisez notre application de jeu de solitaire Spider.

Informations collectées

Données stockées localement

Spider Balala stocke uniquement des données localement sur votre appareil :

  • État de la partie en cours : Votre progression dans la partie actuelle
  • Préférences de jeu : Vos paramètres d'affichage (indicateurs d'aide, etc.)
  • Historique des mouvements : Pour la fonction "Annuler"

Ces données sont stockées dans le stockage local de votre navigateur et ne sont jamais transmises à nos serveurs ou à des tiers.

Données non collectées

Nous ne collectons PAS :

  • Informations personnelles identifiables
  • Coordonnées (email, téléphone, adresse)
  • Données de localisation
  • Accès à la caméra ou au microphone
  • Contacts ou liste d'appels
  • Identifiants publicitaires

Utilisation des données

Les données stockées localement sont utilisées uniquement pour :

  • Sauvegarder votre partie en cours
  • Mémoriser vos préférences de jeu
  • Permettre la fonction "Annuler"

Partage des données

Nous ne partageons aucune donnée avec des tiers. Toutes les données restent sur votre appareil.

Services tiers

L'application n'utilise aucun service tiers d'analyse, de publicité ou de suivi.

Sécurité

Vos données sont stockées de manière sécurisée dans le stockage local de votre navigateur. Seule l'application y a accès.

Droits des utilisateurs

Vous pouvez à tout moment :

  • Supprimer vos données en effaçant les données de l'application dans les paramètres de votre appareil
  • Effacer votre partie en cours en démarrant une nouvelle partie
  • Désinstaller l'application pour supprimer toutes les données

Applications mobiles (TWA - Trusted Web Activity)

Si vous utilisez Spider Balala via une TWA (application Android), les mêmes règles de confidentialité s'appliquent. L'application ne demande aucune permission système et fonctionne comme une application web.

Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps en temps. Toute modification sera publiée sur cette page avec une nouvelle date de mise à jour.

Contact

Si vous avez des questions concernant cette politique de confidentialité, vous pouvez nous contacter via :

  • GitHub : https://github.com/votre-username/spider-solitaire

Conformité RGPD

Spider Balala est conforme au Règlement Général sur la Protection des Données (RGPD) car :

  • Aucune donnée personnelle n'est collectée
  • Aucun traitement de données à caractère personnel n'est effectué
  • Toutes les données restent sur l'appareil de l'utilisateur

Conformité Play Store

Cette application respecte toutes les exigences du Google Play Store concernant :

  • La confidentialité des utilisateurs
  • La protection des données
  • La transparence sur la collecte de données
  • L'absence de pratiques trompeuses

Pour les enfants

Spider Balala est adapté à tous les âges. Aucune donnée personnelle n'est collectée, ce qui rend l'application sûre pour les enfants.

Résumé

Spider Balala est un jeu de solitaire entièrement hors ligne qui ne collecte aucune donnée personnelle. Toutes vos données de jeu restent sur votre appareil et ne sont jamais transmises.
